## 📌 Overview
The **Cyber Attack Simulator** is an interactive environment that simulates real-world cyber attack scenarios involving three main roles:

- 🧑‍💻 Attacker (Hacker)  
- 🖥️ Victim Server  
- 🛡️ Defender  

The goal is to **educate defenders**, improve understanding of cyber threats, and provide a **real-time dashboard** for monitoring and decision-making.

---

## 🎯 Hackathon Context
This project was developed during the **HPI GovTech Student Award Hackathon**.

**Focus areas:**
- Strengthening trust in digital government systems  
- Simulating realistic cyber attack scenarios
